Q1:什么是软件工程？

软件工程是一门研究用工程化方法构建和维护有效的、实用的和高质量的软件的学科。它涉及程序设计语言、数据库、软件开发工具、系统平台、标准、设计模式等方面。

Q1':软件工程的三个要素是什么?

方法 过程 工具

Q2:什么是软件过程？列举四个典型软件过程模型。

软件过程是形成软件产品的一系列步骤，是一个为建造高质量软件所需完成的任务的框架。

列举四种常见模型:

  • 瀑布模型(传统生存周期模型)
  • 原型实现
  • 螺旋模型
  • 增量模型
  • ...

Q3:简述CMM的五个等级

初始级 initial
工作无序 主要依赖个人经验和能力
可重复级 Repetatble
管理制度化 初步实现标准化
已定义级 Defined
开发过程实现了标准化和文档化
已管理级 Managed
产品和过程已建立了定量的质量目标
优化级 Optimizing
可集中精力改进过程，采用新技术新方法，拥有防止出现缺陷，识别薄弱环节以及加以改进的手段。
